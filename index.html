Practice how to pull request!

Steps:

  1. Fork the page to create a personal version your own github page.
  2. Open terminal.
  3. Clone your fork onto your pc (git clone ~paste link to forked repo here~)
  4. Create a new branch with whatever name you like (git branch newBranchName)
  5. Move into that new branch (git checkout newBranchName)
  6. Add your name to the index.html list below with your beloved IDE! *cough*VSCODE SUPREMACY*cough*
  7. After your changes, use the below commands to stage and commit your work:

    8. git add .

    git commit -m "Your descriptive message here, outlining your changes"

    (Don't forget the 'dot' in the git add command above! The dot there means 'add everything')

  8. Now time to push your local branch to the original repo! (git push origin newBranchName)
  9. Now head back to your forked repo page on github.com (refresh the page), you should see a notification saying
  10. Click on it. Add a title for your pull request and a descriptive comment. This will help the owner to see what you done and hopefully approve and merge your changes!
